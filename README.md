# Demo Geoportal Municipal - Inteligencia Territorial

Plataforma de Sistemas de Información Geográfica (SIG) diseñada para la gestión y optimización de recursos públicos a nivel municipal.

## Tecnologías
* **Frontend:** Astro
* **Visor Geográfico:** OpenLayers
* **Despliegue:** GitHub Actions -> Servidor Propio

## Objetivo
Demostrar a las administraciones locales cómo la "Inteligencia Territorial" permite:
- Visualizar el gasto público geolocalizado.
- Planificar infraestructura (ej. hospitales, escuelas).
- Transparentar las obras públicas.
